# Full Stack Developer Portfolio Plan (v2)

*Updated to reflect the new GSAP & Framer Motion driven React architecture.*

## 🎨 Theme & Style
- **Theme:** Dark theme (black/charcoal base with subtle gradients).
- **Accent colors:** Neon blue / purple / teal glow.
- **Aesthetics:** Glassmorphism + soft shadows.
- **Typography:** Minimal, clean modern sans-serif (e.g., Inter, Outfit).
- **Interactions:** Smooth transitions, micro-interactions, and scroll-based animations.

## 🧭 Pages & Sections
1. **Landing / Hero Section.** Full-screen hero, animated particles/waves background. GSAP intro animation (text reveal, fade, parallax). Tagline, CTA buttons.
2. **About Section.** Brief intro. Skills categorized (Frontend, Backend, Tools). Framer Motion hover animations on skill cards.
3. **Projects Section.** Hover effects (scale, glow, tilt). Tech stack filters. Smooth scroll reveal animations.
4. **Pricing Section.** Three transparent tiers (Basic, Standard, Premium) with Indian Rupee (₹) pricing. Animated hover glows.
5. **Services Section.** Icon-based layout for Web Dev, APIs, Automation, Custom Software.
6. **Testimonials.** Animated carousel.
7. **Contact Section.** Name, Email, Message with focus animations. Social links.

## ⚡ Animations & Interactions
- **GSAP:** Scroll-triggered animations, Hero section intro, Parallax.
- **Framer Motion:** Component transitions, Hover/tap interactions, Smooth page transitions.

## 💻 Tech Requirements
- **Framework:** React or Next.js.
- **Styling/Animations:** Tailwind CSS (recommended for this style), GSAP, Framer Motion.
- **Features:** Custom cursor, loading screen, sticky navbar. Responsive & SEO-friendly.
